## Equipo técnico requerido

### Cámara
- [ ] Cámara principal (Sony FX3, FX6, Canon C70 o equivalente full-frame)
- [ ] Cámara B / cámara documental (Sony A7S, Canon R5 o similar)
- [ ] Drone (opcional para tomas aéreas Granada) — verificar permiso AESA
- [ ] Gimbal o estabilizador para tomas en movimiento

### Óptica
- [ ] Zoom 24-70mm f/2.8 o 24-105mm f/4 (versatilidad)
- [ ] Prime 35mm o 50mm f/1.8 (planos cálidos íntimos)
- [ ] Tele 70-200mm f/2.8 (cobertura sin invadir)

### Audio
- [ ] Grabadora de audio externa (Zoom H6, Tascam DR-60 o similar)
- [ ] Lavalier inalámbrico para testimonio (Rode Wireless Go II, Sennheiser EW-DP)
- [ ] Boom + caña + jirafa (para testimonio sin lavalier visible si toca)
- [ ] Auriculares de monitoreo

### Iluminación
- [ ] Panel LED bicolor para interior (Aputure 600d, Godox SL150 o similar)
- [ ] 2-3 paneles pequeños para fill y contraluz (Aputure MC, Falcon Eyes)
- [ ] Difusores y banderas (control de luz natural)
- [ ] Reflector 5-en-1 (sol)

### Soporte
- [ ] Trípodes (2 mínimo)
- [ ] Slider o dolly ligero (movimientos lentos en planos producto)
- [ ] Hi-hat o trípode bajo para planos detalle
