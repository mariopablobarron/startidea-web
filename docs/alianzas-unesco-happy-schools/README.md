# Alianzas UNESCO Happy Schools — outreach a fundaciones

Documentación operativa del plan para financiar el programa **UNESCO Happy Schools × Design for Change** mediante alianzas con fundaciones grandes españolas e internacionales.

## Contexto rápido

- Programa avalado por UNESCO (Happy Schools Agnostic Toolkit) y operado por Design for Change en 35 países / 4 continentes.
- Mario Pablo Sánchez Barrón (Startidea) actúa como representante global, nombrado por Kiran Bir Sethi (fundadora DFC).
- UNESCO presta marca y mide impacto; no aporta dinero directo.
- Objetivo de financiación: **€1M–€5M / año** vía 3-5 fundaciones co-financiando.
- Líneas a financiar: coordinación internacional + plataforma digital con IA + formación de facilitadores + medición de impacto.

## Estructura

| Archivo | Propósito |
|---|---|
| `00-plan-estrategico.md` | Objetivos, KPIs, calendario realista, equipo |
| `01-pitch-one-pager.md` | One-pager que se manda a la fundación. Incluye placeholders `{{UNESCO_X}}` para los datos del toolkit oficial |
| `02-fundaciones-objetivo.md` | Lista priorizada de 25-30 fundaciones con criterio de encaje |
| `03-email-outreach-templates.md` | Plantillas de primer contacto + variaciones según tipo de fundación |
| `04-flujo-seguimiento.md` | Funnel: primer contacto → reunión → propuesta → cierre. Respuestas tipo |

## Cómo usar

1. **Antes de mandar nada**: completar los `{{UNESCO_X}}` del pitch con datos reales del Happy Schools Toolkit y del estudio UCJC.
2. Decidir las primeras 5 fundaciones a las que se manda outreach (de la lista en `02-`).
3. Personalizar el email de `03-` con el lenguaje específico de cada fundación.
4. Llevar registro en CRM (incluso una hoja de cálculo basta) con fechas, respuestas, próximas acciones — el flujo está en `04-`.
